About the Project

Medical Insurance Cost Prediction using Machine Learning

Project Overview

Medical insurance companies estimate insurance charges based on several factors such as age, gender, body mass index (BMI), number of children, smoking status, and residential region. This project applies machine learning techniques to predict annual medical insurance costs accurately using patient information.

Problem Statement

Manual estimation of insurance costs can be inconsistent and time-consuming. This system automates the prediction process using machine learning to provide quick and reliable estimates.

Project Objectives

  • Predict annual medical insurance costs.
  • Compare multiple regression algorithms.
  • Select the best-performing machine learning model.
  • Provide predictions through a user-friendly web application.
  • Help users understand factors influencing insurance costs.

Dataset Information

  • Total Records: 1338
  • Input Features: 8
  • Target Variable: Insurance Charges
  • Age
  • Sex
  • BMI
  • Children
  • Smoker
  • Region

Machine Learning Workflow

Dataset ↓ Data Preprocessing ↓ Feature Engineering ↓ Model Training ↓ Model Evaluation ↓ CatBoost Model ↓ Prediction

Why CatBoost?

Several regression models were evaluated, including Random Forest, Gradient Boosting, XGBoost, and CatBoost. CatBoost achieved one of the highest prediction accuracies while maintaining strong cross-validation performance, making it the selected model for deployment in this application.
